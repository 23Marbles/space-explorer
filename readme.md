# Space Colonizer
#### _my submission to godot wild jam 90_

## Overview

The premise of the game is to explore space searching for planets to colonize and live on. You map a trail behind you as you travel which is revealed at the end.

## Overview

### Gameplay loop
```
Blast into space
   |
   |
Place stations in orbit above planets
   |                   |
   |                   |
Link stations ———— Find new planets
   |
   |
View your network
```
